    1.3 awk取行

    • 与sed类似

    1） 案例01 取出sed.txt的第2行

     [root@dog /oldboy]# cat sed.txt
101, oldboy,CEO
102, zhangya, CTO
103, lidao007,COO
104, yy,CFO
105, feixue, CIO
110, lidao, COCo
 [root@dog /oldboy]# awk 'NR==2' sed.txt
102, zhangya, CTO
 [root@dog /oldboy]# #awk '行号 等于 2' sed.txt


#NR awk内置变量
Number of Record 记录号(行号)

    2）案例02 取出sed.txt中包含oldboy或lidao的行

    [root@dog /oldboy]# sed -rn '/oldboy|lidao/p' sed.txt
101, oldboy,CEO
103, lidao007,COO
110, lidao, COCo
 [root@dog /oldboy]# awk '/oldboy|lidao/' sed.txt
101, oldboy,CEO

110, lidao, COCo

    3）案例03 取出文件第2行和第5行

     # awk ‘行号大于等于2 并且 行号小于等于5’ sed.txt
 
 #方法01
 awk 'NR>=2 && NR<=5' sed.txt  #必会
 #方法02
 sed -n '2,5p' sed.txt
 
 #方法03 
 awk 'NR==2,NR==5' sed.txt #从行号是2的行开始 到 行号是5的行结束（了解）
 
 
[root@dog /oldboy]# awk 'NR>=2 && NR<=5' sed.txt

    1.4 awk取列

    • 目标：熟练取出指定的内容或不服（取列）
    #awk取列 
awk '{print $1,$3}'

    1)案例01 取出ls -lh命令中的第1列和第3列

    ll -h |awk '{print $1,$3}'

    注意 awk中 ‘$数字’只有一个意思，某一列（取列）

    2）案例02 取出/etc/passwd每一行内容，加上行号

    #方法01 
cat -n /etc/passwd

#方法02
awk '{print NR，$0}' /etc/passwd

NR表示行号
$0 表示一整行内容

    3) 案例03 取出/etc/passwd第1列和第3列

    • awk取列的，默认以空格，连续空格或tab键空格分割，只需要使用$1.$2,$6
    • 如果想更改或指定新的，分隔符，需要使用 -F 选项
    • -F指定分隔符，-F支持正则
    awk -F':' '{print $1,$3}' /etc/passwd | column  -t

    4）案例04 取出ip.txt文件中ip（10.0.0.200）地址

    • awk中指定多个分隔符，需要使用正则匹配这多个分隔符即可
    • -F’正则’
    echo '    inet 10.0.0.200/24 brd 10.0.0.255 scope global ens33' >/oldboy/ip.txt

    #方法01 多个管道

    5）案例 取出/etc/passwd第1列第3列以及最后一列

    $NF 最后一列

    NF Number of Filed 每行有多少个字段(域，列) 每行有多少列

    
awk -F':' '{print $1,$2,$NF}' /etc/passwd

    6）awk取列小结

    • 一般配合’{print NR,$1,$2,$NF}’
    • NR行号，$内容，某一列
    • $NF最后一列
    • $0表示整行
    • 案例
      • 取默认的列
      • 取指定分隔符，指定分隔符使用正确
    • 取列核心：使用-F指定分隔符
    • 李导取分隔符建议：以你目标两边的内容作为分隔符即可，可以是单个字符，也可以是单词

    1.5 awk行与列综合（awk过滤进阶）

    1）精确过滤必备

    • awk可以完成，某一列中包含/不包含内容
    • $3 ~ // 匹配或包含
    • $3 !~ // 不匹配或不包含
    格式：
awk -F:'第3列 包含 以1或2开头的内容' passwd
#示例：取出passwd中第3列以1或2开头的内容，显示着一行
awk -F:'$3 ~ /^[12]/' passwd

    2）案例01 过滤出passwd中第3列以0到3结尾的，显示第1列和第3列内容

    条件：第3列以0到3结尾
动作：显示第1列和第3列（取列）

#方法01 分步骤
awk -F: '$3~/[0-3]$/' /etc/passwd
awk -F: '$3~/[0-3]$/' /etc/passwd | awk -F: '{print $1,$3}'

#方法02 awk同时取行与取列
awk -F: '$3~/[0-3]$/{print $1,$3}' /etc/passwd

    3）案例02 过滤passwd中第3列大于0小于1000内容，显示第1列和第3列

    条件(取行)：过滤passwd中第3列大于0小于1000内容
动作(取列)：显示第1列和第3列
awk -F: '$3>0 && $3<1000{print $1,$3}' /etc/passwd

    1.6 awk统计计算功能

    1）BEGIN{}和END{}

    awk中的特殊的条件(模式)，pattern

    这两个特殊的条件，斗鱼计算相关，最常用的是END{}

    特殊条件（模式） 含义 应用场景
    BEGIN{} BEGIN{}内容会在awk读取文件之前执行 1.里面进行实验
    2.进行计算，不需要读取文件内容
    3.创建或修改awk变量
    END{} END{}内容会在awk读取文件之后执行 1.awk在读取文件的时候进行统计与计算，最后统计完成在END{}输出最终结果

    2）BEGIN{}了解

    #计算 1/3
+ - * /
 [root@dog /oldboy]# awk 'BEGIN{ print 1/3,1-3,1+3,3*3,2^10}'
0.333333 -2 4 9 1024

#修改awk变量
awk -F: 'NR==1{print $1}' passwd

awk 'BEGIN{FS=":"} NR=1{print $1}'

    3)END{}熟练掌握

    a}案例01 统计/etc/passwd的行数

    wc -l /etc/passwd
#条件：无
#动作：i++  #统计次数，类似于wc -l
awk '{i++} /etc/passwd
#输出最后i内容
awk '{i++} END{print i}' /etc/passwd



#输出最后i内容
